Data analysis project exploring laptop pricing using Python. Includes data cleaning, EDA, and insights on how features like RAM, brand, and type affect price.

# 💻 Laptop Price Analysis

This is a beginner-friendly data analysis project where I explored a laptop dataset to understand what actually affects laptop prices.

Instead of just running code, I tried to think like a data analyst — cleaning messy data, exploring patterns, and finding meaningful insights.
## 🔍 What I did in this project

* Loaded and explored the dataset to understand the structure
* Cleaned the data (handled missing values, removed unnecessary columns)
* Converted columns like RAM and Weight into proper numeric format
* Explored price distribution using charts
* Compared price across different companies and laptop types
* Analyzed how RAM and other features affect price
* Used correlation to understand relationships between variables
## 📊 Key Insights

* Most laptops fall in the mid-price range, while a few premium models push the prices higher. This creates a right-skewed distribution.
* RAM clearly plays a major role in pricing. As RAM increases, the price also increases significantly.
* Weight, on the other hand, does not seem to have much impact on price.
* Brands like Razer are positioned in the premium segment, while brands like Vero focus on budget laptops.
* Workstation and gaming laptops are generally more expensive due to high performance, while notebooks and netbooks are more affordable.
* 8GB RAM laptops appear most frequently in the dataset, showing that it is a common choice for users.
* Price does not increase evenly with RAM — higher configurations show bigger jumps due to premium specifications.
## 🛠️ Tools Used

* Python
* Pandas
